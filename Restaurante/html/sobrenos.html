Nosso Trabalho

Este é um trabalho acadêmico desenvolvido para a disciplina de Autora Web, ministrada pelo professor Fabrício ___ no Instituto Federal de Educação, Ciências e Tecnologia do Rio Grande do Norte.


O trabalho consistiu na criação de um site utilizando as linguagens de programação HTML e CSS, com a possibilidade de incluir também JavaScript. O tema escolhido foi um restaurante, e decidimos criar um restaurante nordestino. Para isso, elaboramos um nome, uma logo e um cardápio com comidas típicas da região


Para a realização do projeto, foram criadas páginas em HTML, incluindo index, cardápio, pedido, avaliação, sobre nós, login e esqueceu a senha. Essas páginas foram estruturadas com HTML, estilizadas com CSS e contaram com algumas funções implementadas em JavaScript.


Após a elaboração, o site foi vinculado ao GitHub
